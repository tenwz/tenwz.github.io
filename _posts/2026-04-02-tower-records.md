## 一、创始人：罗素·所罗门（Russ Solomon）

### 从药房柜台到音乐帝国

1922年，罗素·所罗门出生于美国加利福尼亚州萨克拉门托。1941年，年仅16岁的他，在父亲经营的 Tower Drugstore（塔楼药房）里，发现了一个商机。

当时药房有一个点唱机（Jukebox），当唱片需要更换时，所罗门建议父亲把替换下来的二手唱片卖掉。他们以3美分一张收购，10美分一张卖出，结果销量出奇地好。几周后，他们开始进新唱片卖，并在药房角落里划出一张桌子大小的空间作为唱片柜台。

这就是 Tower Records 的雏形。

所罗门后来在纪录片《All Things Must Pass》中回忆：「我父亲想让我成为药剂师，但我不想。卖唱片比卖药有趣多了！」

1960年，已经退伍并经历过一次批发业务失败的所罗门，借钱在萨克拉门托 Watt 大道开设了第一家独立的 Tower Records 门店，约100平方米。店名延续自父亲的药房和隔壁的 Tower Theatre（塔楼剧院）。

2018年3月4日，92岁的所罗门在萨克拉门托家中观看奥斯卡颁奖典礼时，让妻子给他续一杯威士忌，妻子回来时他已经因心脏病安详离世。据他儿子迈克尔说，他当时正在评论某个人的着装很难看。

## 二、运营哲学：买手制与去中心化

### 1. "如果我不喜欢它，我为什么要卖它？"

这是所罗门的核心信条。他要求每一位店员都必须是音乐爱好者，甚至是某一音乐流派的专家。

**买手制（Buyer System）** 是 Tower Records 最核心的运营机制：

- **店员即买手**：每个店员不只是理货员，而是有权决定店内进什么货的采购者
- **本地化选品**：每家店根据当地顾客的品位独立选品，新奥尔良的店有庞大的传统音乐区，纳什维尔的店有巨大的乡村音乐区
- **从仓库到经理**：公司所有人几乎都是从基层做起，今天在后清点货的员工，明天可能成为仓储物流经理

所罗门说：「找到那些能把事情做好的人，不管他们玩得多开心，然后别管他们。」这是一种危险但有效的放任式管理。

### 2. 没有制服的自由

Tower Records 的员工可以留长发、穿破洞牛仔裤，没有统一制服。所罗门认为，员工的"酷劲儿"和对音乐的了解会吸引顾客。

纪录片导演科林·汉克斯说：「Tower 本质上是一群独立唱片店的集合……每家店都代表它所在的城市或社区，有自己的风格。」

## 三、空间设计与分类学

### 1. 唱片超市：把唱片堆在地板上

所罗门将 Tower Records 定位为"音乐超市"（Record Supermarket）：

- **商品堆在地板上**：像超市一样，唱片直接堆在地上，营造琳琅满目的感觉
- **地板到天花板的陈列**：货架尽可能高，填满每一寸空间
- **午夜营业**：早上9点到午夜12点全年无休，在网购时代之前，这是音乐爱好者的深夜避难所

### 2. 深度分类学

Tower Records 的分类极其细致：

```
摇滚 → 另类摇滚 → 90年代英国另类摇滚 → 具体乐队
```

这种深度分类不仅方便检索，更是一种价值表达：分类的深度直接告诉顾客"我们有多重视这个领域"。

### 3. 试听站（Listening Stations）

2005年，Tower 引入了"扫描即听"（Scan and Listen）站台。顾客可以试听 CD 的音频样本，搜索特定歌曲、专辑和艺人。这降低了在风格多样的音乐中做选择的难度，提升了消费体验。

### 4. 标志性设计

- **黄色与红色的 Logo**：灵感据说来自 Shell 石油的标志。所罗门开玩笑说：「不是'灵感'，是'盗用'，我们偷来的。」设计师是 Charles Nicholson，选择这个配色是因为"远远就能看见"
- **黄色购物袋**：成为文化符号
- **"NO MUSIC, NO LIFE"**：日本店创造的 Slogan，成为 Tower 的"第二 Logo"

## 四、扩张与全球化

### 关键节点

| 年份 | 事件 |
|------|------|
| 1941 | 16岁的所罗门在父亲药房开始卖二手唱片 |
| 1960 | 第一家独立 Tower Records 在萨克拉门托开业 |
| 1968 | 旧金山店开业，200坪，成为全美最大唱片店 |
| 1970 | 好莱坞日落大道店开业，成为音乐人的聚集地 |
| 1980 | 日本札幌店开业，是第一家海外店 |
| 1983 | 创办《Pulse!》杂志，免费发放，1992年开始全国发行 |
| 1995 | 与 Good Guys 合作推出电子产品超商 Wow! |
| 1999 | 年销售额突破10亿美元，全球约200家店 |
| 2004 | 申请破产保护 |
| 2006 | 美国所有门店结业 |
| 2020 | 以线上商店形式回归 |

### 日本：最成功的海外市场

日本是 Tower Records 在全球最成功的市场，甚至超过美国本土。

1980年，两个日本人来到萨克拉门托的 Tower 总部，希望在日本开店。所罗门先以批发业务试水，在东京赤坂某大厦7楼设仓库，但一年后发现自己并不擅长批发。于是他决定直接开零售店，在涩谷找到了一个约150坪的店面。

开业即爆发。所罗门回忆说：「当时全是黑胶，我们每天都从加州往日本发货，太棒了。」

如今，日本 Tower Records 在东京涩谷拥有8层大楼的总店，面积52,000平方英尺，是世界上最大的音乐零售店。2016年，仅涩谷店平均每天就有3-4场活动（小型演唱会、见面会、签名会）。

日本店创造的 "NO MUSIC, NO LIFE" 口号，成为 Tower 全球性的文化符号。

## 五、营销与社区建设

### 1. 店内演出与奇思妙想

- **停车场演唱会**：在旧金山店举办过杰克逊五兄弟的演出
- **醉象促销**：为推广 The Band 的专辑《Music from Big Pink》， Capitol 唱片的人带了一头被香槟灌醉的粉象进店
- **棺材促销**：Grateful Dead 新专辑发行时，华纳的宣传人员把自己装在棺材里抬进店

### 2. 名人效应

- **埃尔顿·约翰**：曾夸口说"我在 Tower 花的钱比任何人都多"
- **好莱坞日落大道店**：地理位置优越，成为巨星和乐手经常光顾的地方，也是当地年轻人约会见面的地标
- **乡村音乐明星**：纳什维尔店靠近音乐街（Music Row），常有乡村音乐明星表演或购物

### 3. 《Pulse!》杂志

1983年创办，最初免费发放以促销唱片销售。1992年开始全国发行，售价2.95美元。杂志包含唱片评论、艺人访谈和广告，至今仍在出版。

## 六、衰落与重生

### 衰落原因

1. **过度扩张**：90年代海外扩张过于激进，欠下超过3亿美元银行贷款
2. **数字革命**：Napster 等文件分享服务让音乐免费获取，CD 销量断崖式下跌
3. **大卖场竞争**：Best Buy、沃尔玛等大卖场以音乐为引流产品，亏本销售 CD
4. **互联网与亚马逊**：网购让顾客无需出门就能买到唱片

所罗门在1997年就预感到危机，他说：「行业必须适应变化，我们正在失去年轻人。」

### 破产

- 2004年：申请第11章破产保护
- 2006年8月20日：美国所有门店结业

有趣的是，所罗门在 Tower 结业几个月后，在萨克拉门托原址附近开了一家小型独立唱片店 R5，但三年后卖给了当地竞争对手。

### 重生

2020年11月，Tower Records 以线上唱片店的形式回归，销售黑胶唱片和周边产品，并计划不定期做线下快闪店（Pop-Up Shop）。新任 CEO Danny Zeijdel 曾是 Tower 的长期顾客，他认为「随着 Tower Records 的结束，音乐行业的一个篇章也结束了，那种社区意识消失了。」

## 七、价值与启示

### 1. 员工赋权

Tower Records 的成功证明：给予一线员工决策权，让他们成为所在领域的专家，比中央集权式管理更有效。

### 2. 本地化与个性

每家 Tower 店都有自己的个性，反映所在城市的音乐文化。这种"本地化连锁"模式，让大集团拥有小店的灵魂。

### 3. 空间即内容

Tower 不仅卖唱片，更创造了一个音乐爱好者可以待上几个小时的空间。试听站、深夜营业、分类深度、员工的专业知识——所有这些都让逛唱片店成为一种体验，而不仅仅是交易。

### 4. 从货架到内容平台

Tower Records 的机制对今天的"内容平台"设计仍有启示：

- **买手制** → 策展人制度
- **深度分类学** → 精细化内容分类
- **员工手写推荐卡** → 编辑推荐/算法推荐
- **试听站** → 内容预览

正如那部纪录片的标题所说：万物皆有尽时（All Things Must Pass）。但 Tower Records 留下的遗产——对音乐的热爱、对员工的信任、对顾客的尊重——仍在影响着我们如何发现和消费文化内容。
